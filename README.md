# Xiaolong Han Personal Homepage

Source for the GitHub Pages site at <https://xiaolonghan2000.github.io>.

## Structure

- `_pages/about.md`: main homepage content
- `_config.yml`: site metadata and author sidebar information
- `_includes/author-profile.html`: sidebar profile rendering
- `images/profile.jpg`: profile photo

## Local preview

```bash
bundle install
bundle exec jekyll serve
```

The site will then be available at `http://127.0.0.1:4000`.
